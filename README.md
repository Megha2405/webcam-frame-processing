# webcam-frame-processing

## 📌 Project Overview
This project demonstrates how to use OpenCV in Python for image capture and real-time video processing using a webcam. The application captures frames from the webcam and performs different operations such as saving images, displaying live video, resizing frames, and rotating the video stream.

The project is designed for beginners to understand the fundamentals of computer vision and video manipulation using OpenCV.

---

## 🎯 Objectives
- Capture images from a webcam
- Save captured frames as JPG files
- Display live webcam video
- Resize video frames dynamically
- Rotate video frames in real time

---

## 🛠️ Technologies Used
- Python 3.7
- OpenCV (`cv2`)
- NumPy
- Jupyter Notebook / VS Code
- Anaconda

---

## ⚙️ Working Process

### 1. Webcam Initialization
The webcam is accessed using OpenCV’s `VideoCapture()` function.

### 2. Frame Capture
Video frames are continuously captured from the webcam.

### 3. Image Saving
A selected frame is saved as a JPG image using `cv2.imwrite()`.

### 4. Video Display
The live webcam feed is displayed using `cv2.imshow()`.

### 5. Frame Resizing
Frames are resized to a custom resolution for better visualization.

### 6. Video Rotation
The video stream is rotated using OpenCV rotation functions.
